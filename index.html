Sincronización entre dispositivos

Conectá Firebase para que tus datos se sincronicen automáticamente entre el celular y la PC.

Sincronizado
Cómo configurar Firebase →
1. Abrí console.firebase.google.com
2. Creá un proyecto → Agregar app web
3. Copiá el objeto firebaseConfig y pegalo arriba
4. Activá Authentication → Google
5. Creá Firestore Database (modo producción)
6. En Reglas de Firestore, pegá esto: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{uid} {
      allow read, write: if request.auth.uid == uid;
    }
  }
}
7. Configurá tu dominio en Authentication → Authorized domains
8. Conectá desde este dispositivo → iniciá sesión con Google
9. Repetí en el otro dispositivo con el mismo config → la sesión queda vinculada
